About the Application

This web application is a web based audio/music Visualizer utilizing the JavaScript canvas API in order to create responsive animations. Along with visual attributes that enhance your listening experience, this audio visualizer also has the ability to augment the audio itself which is possible through toggling with the controls.

Audio Effects: The two audio effects I added to my visualizer were a treble booster and a bass booster that can be toggled via checkboxes on the side control panel.
